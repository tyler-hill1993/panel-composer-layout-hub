## What Panel Composer Does

Panel Composer is built for assembling separate panels into a clean, publication-ready figure without forcing you to rebuild every layout by hand. It is meant for workflows that combine PDF pages with common image formats, so you can merge diverse source material into a single structured composition with less friction.

This tool fits researchers, designers, and anyone who prepares multi-part visual layouts. It brings together AI-assisted arrangement and hands-on canvas editing, letting you either start with an automatic suggestion or fine-tune the final result through drag-and-drop controls and natural language instructions.

## Key Capabilities

- Merges multiple panels into a single figure
- Handles mixed PDF and image inputs
- Provides AI-assisted automatic layout
- Supports drag-and-drop placement on an HTML canvas
- Accepts natural language layout instructions
- Keeps aspect ratio intact to prevent distortion
- Applies automatic A/B/C or numeric labels
- Exports as vector PDF or raster PNG

## How to Use It

A common workflow is to bring in your source panels, place them on the canvas, tune spacing and labels, and then export the finished figure in the format you need.

Example workflow:
1. Import PDF pages, PNG, JPG, TIFF, or BMP files.
2. Use drag-and-drop on the canvas to position each panel.
3. Apply automatic layout if you want a fast first draft.
4. Refine the arrangement with natural language layout instructions.
5. Add panel labels such as A, B, C, or numbered markers.
6. Export the finished composition as PDF or PNG.

For repeatable work, keep your panel order and export settings consistent across figures.
